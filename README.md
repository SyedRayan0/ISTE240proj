# EventHub Web-Based Event Organizer Platform

EventHub is a web-based subscription platform designed for event attendees to discover nearby events while enabling administrators and organizers to create and promote events through a centralized system. The platform allows users to browse and subscribe to events, receive live notifications and updates, and stay informed about upcoming activities, while organizers and admins can manage events and monitor participation metrics through an interactive dashboard. **Team Members:** Syed Rayan Hashmi (ID: 764000072), Ali Faisal (ID: 415002117).
